# Spec Hunt — Templates

Reusable templates for starting new engagements. These are used internally by `init.sh` and `new_finding.sh` — you don't need to copy them manually.

## Templates

| Template | Purpose |
|----------|---------|
| `CLAUDE.md.template` | Agent instructions for the engagement |
| `status.md.template` | Engagement-level status tracker |
| `scope.md.template` | Structured scope summary |
| `finding.md.template` | Per-finding metadata and state |
| `*_README.md` | README templates for each folder |
